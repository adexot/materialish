## Usage

```jsx
import { Checkbox } from "materialish";

export default function() {
  return (
    <Checkbox>
      Check me
    </Checkbox>
  );
}
```

## Props

| Prop Name | Default Value | Required | Description                                   |
| --------- | ------------- | -------- | --------------------------------------------- |
| alignment  | "left"              | No       | Whether the label appears to the left or the right                |

## Tips

* The Checkbox is useful for X, Y, Z
