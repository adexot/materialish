## Usage

```jsx
import { Spinner } from 'materialish';
import 'materialish/spinner.css';
```

## Props

| Prop Name | Default Value | Required | Description                                      |
| --------- | ------------- | -------- | ------------------------------------------------ |
| size      | "1rem"        | No       | The size of the Spinner                          |
| ...rest   |               | No       | Other props are placed on the root `svg` element |
